# 🌍 LankaGo AI – Smart Sri Lanka Travel Planner

LankaGo AI is a modern full-stack travel planning web application designed to help users create personalized travel plans across Sri Lanka using Artificial Intelligence.

The system provides smart itinerary generation, weather warnings, budget optimization, hotel recommendations, and an AI travel assistant — all inside a responsive and professional web platform.

---

# ✨ Features

## 🔐 Authentication System
- User Registration & Login
- Secure Password Hashing
- Session Management
- Profile Management

## 🤖 AI Trip Planner
Users can generate travel itineraries based on:
- Budget
- Number of Days
- Travel Destinations
- Travel Style

The AI generates:
- Daily travel plans
- Suggested activities
- Estimated travel cost
- Hotel recommendations

## 💰 Smart Budget Optimizer
The system intelligently adjusts:
- Hotels
- Transport methods
- Activities

according to the user's budget.

## 🌦️ Weather Warning System
- Real-time weather updates
- Rain alerts
- Temperature forecasts
- Travel condition notifications

## 💬 AI Travel Assistant
An integrated chatbot powered by OpenAI API helps users with:
- Travel recommendations
- Destination suggestions
- Budget advice
- Trip-related questions

## 🗺️ Google Maps Integration
- Route visualization
- Nearby attractions
- Travel distance estimation
- Interactive maps

## ❤️ Saved Trips
Users can:
- Save itineraries
- Edit plans
- Manage travel history

## 📱 Fully Responsive Design
Optimized for:
- Mobile devices
- Tablets
- Desktop screens
